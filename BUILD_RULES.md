# AtoZee Website Build Rules

## Goal
Build a premium dark-theme B2B technology website for AtoZee Solutions inspired by the UI quality, spacing, and composition of modal.com.

## Core Rules
- Do not copy modal.com branding, text, or assets
- Only use it as visual inspiration
- Keep design premium, minimal, and modern
- Use reusable React components
- Use Next.js App Router
- Use Tailwind CSS
- Use Framer Motion for subtle animation only
- Use Inter as the primary font
- Keep code clean and production-ready
- Maintain mobile responsiveness
- Preserve consistent spacing, typography, and visual rhythm

## Design Rules
- Dark background
- High contrast typography
- Large bold hero headings
- Rounded pill buttons
- Glassy floating navbar
- Subtle gradients and glow
- Strong centered layouts
- Spacious sections
- Minimal clutter

## Coding Rules
- Edit only requested files
- Do not break existing components
- Reuse utility classes when possible
- Keep components modular
- Avoid unnecessary dependencies
- Prefer lightweight CSS + Framer Motion over heavy 3D libraries unless required
